About Page

Hello my name is Jose Miranda, and I created this uno game as a part of my capstone project to test my skills. This Uno game was built to challenge myself and because I wanted to create something fun I can look back on.

This project is a browser game based on the game Uno. Built using:

  • HTML5
  • CSS3
  • JavaScript

The game uses Javascript features such as:

  • Functions to control gameplay
  • Arrays to manage the deck
  • Loops to enforce rules
